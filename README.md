# 卡牌塔防最小原型

HTML + Canvas + Vanilla JS 零依赖的小型卡牌塔防原型。玩家在 21×13 迷雾地图上守住中央核心，撑过两次血月、击败终极 BOSS 即视为胜利。

## 如何运行

直接用浏览器（Chrome / Safari / Firefox）打开 `index.html` 即可，脚本全部为普通 `<script>`，支持 `file://` 协议。

如果浏览器策略拦截了本地脚本，启一个本地服务器：

```bash
cd /Users/hudi/Downloads/卡牌塔防原型
python3 -m http.server 8080
# 访问 http://localhost:8080/
```

## 一局节奏（约 15 分钟）

| 天数 | 关键事件 |
|---|---|
| Day 1 | 教学夜（不刷新虫巢），先铺经济 |
| Day 2 起 | 每晚刷新一座新虫巢（25% 几率钢壳），落地立即满兵 |
| Day 3 dusk | 第一次血月：动态补刷至少 1 座钢壳 + 第一巢混出血月 BOSS（300 hp）|
| Day 6 dusk | 第二次血月：再次动态补刷 |
| Day 6 dawn | 撑过两次血月 → 在核心 4–6 格圈外刷出**终极 BOSS**（600 hp / 25 dmg） |
| 终极 BOSS 死亡 | 胜利 |
| Day 9 | 若没及时击杀终极 BOSS，第三次血月加压 |

## 核心系统

### 虫巢
- **普通巢**（80 hp）/ **钢壳巢**（160 hp，白天减伤更高、孵化更快、掉钢壳战利品 3 选 1）
- 开局 2 个完全随机位置（最小距核心 4 格、互距 4 格）
- 每晚 night 阶段补刷一座（最多同时 8 座；血月夜按当前数量动态多刷至少 1 座钢壳）
- **被英雄/箭塔攻击时加速孵化**：当前 hp 越低，下一只虫子来得越快（≥50% → ≤1.5s / <50% → ≤0.8s / <25% → ≤0.3s）

### 虫子
- 普通 / 重甲 / 守卫（黎明绕巢）/ 血月 BOSS（300 hp）/ 终极 BOSS（600 hp，永久揭雾、不集结、直奔核心）
- **进化因子**：随击杀和天数累积放大虫子 hp 与攻击（每杀 +0.2% / 重甲 +0.5% / Boss +5% / 虫巢 +5% / 每天 +10%；上限 ×3）

### 英雄（汤姆）
- 默认在核心 3×3 巡逻，3 格视野内见敌即 ENGAGE，离核心超过 5 格自动回防
- 核心受击瞬间触发归位 + 顶部红色 ❗ 提示
- 开局/复活立即在右下技能槽给一张 **扫击**（半径 1.5、伤害 30、CD 25s）
- 出征攻打虫巢需消耗 1 令牌；行军中不响应核心警报（自行决定是否打"召回令"）

### 剑士
- 兵营生产、维持 2 名；兵营被毁后剑士以核心为锚点继续巡逻
- 与英雄同构状态机（巡逻 / 追击 / 攻击 / 回防）

### 迷雾（驱散 vs 点亮）
- **未驱散区域无法放置建筑**
- **永久驱散**：瞭望塔（半径 3，被毁回滚）/ 探照灯（90° 锥形 1.5s 扫一格，**被毁不回滚** —— 用完可拆）
- **临时点亮**：侦察兵法术（30s 沿途 + 抵达）/ 英雄过雾（5s 周边十字）
- 迷雾里的虫子攻击瞬间：该格红色脉冲圈点亮 2.5s，箭塔可短暂索敌反击

### 仇恨机制
- 瞭望塔 / 探照灯 / 侦察兵 标 `attractsAggro`，被虫子的搜敌范围 ×2 倍优先锁定
- 故意放在外围当"诱饵"可以保护核心和经济建筑

### 卡牌与商店
- 顶部商店栏 7 张普通卡常驻直购：`采胶器 15 / 箭塔 25 / 剑士营 30 / 探照灯 25 / 火雨 20 / 加急 18 / 侦察兵 25`
- 稀有卡（5 张）/ 史诗卡（2 张）只能从战利品包中获得（虫巢击破礼盒 + 血月奖励包）
- 同名手牌自动叠放（×N 徽章），手牌堆数上限 10
- **3 件齐发礼盒**：胶 + 宝石 + 稀有卡（钢壳虫巢的卡走 3 选 1 浮层）

### 天赋树（24 节点 / 4 大类）
- 顶栏 🌟 按钮 或 `T` 键打开
- 点数获取：每天清晨 +1 / 每杀虫巢 +1 / 每 5 个虫子 +1
- 4 大类：
  - **英雄**（9）：锋利 I/II、坚韧 I/II、迅捷、视野、远征、扫击精进、巢穴克星
  - **公共**（5）：所有建筑通用 —— 坚固结构 I/II（hp）、紧急维修（回血）、资源涌泉（每天 +30 胶）、远视联防（射程 +0.5）
  - **军事**（5）：箭塔精进 I/II、剑士营强化、法师精纯、锐利地刺
  - **生产**（5）：采胶丰收、工坊速热、望远千里、灯火通明、远程斥候
- 点亮已建建筑会按比例同步升级 hp（不会回满，保留伤痕）

### 探索奖励（地图宝藏）
- 开局在远离核心的位置撒 4 个金色宝箱
- 玩家单位走过自动拾取，或左键点击宝箱直接拾取（仅在已驱散区域可见）
- 奖励池：胶质 / 宝石 / 稀有卡

## 操作

| 操作 | 键位 / 鼠标 |
|---|---|
| 暂停 / 继续 | 空格 |
| **时间加倍 ×1 → ×2 → ×3** | 顶栏 ⏩ 按钮 / `F` 键 |
| 直购普通卡 | 左键点顶栏商店按钮 |
| 打出手牌 | 左键点手牌堆（同名叠放） |
| **数字键打卡** | `1`–`9` / `0`（对应手牌前 10 堆） |
| 释放扫击 | 左键点右下角技能槽 |
| 天赋面板 | `T` 键 / 顶栏 🌟 按钮 / `ESC` 关闭 |
| 落点确认 | 左键点合法地块 |
| 探照灯方向 | 落点后选 ↑↓←→ 四向之一 |
| 取消放置 | 右键 / `ESC` |
| 出征 | 左键点虫巢 → 确认（消耗 1 令牌） |
| 拾取礼盒 / 宝藏 | 左键点击 或 单位走近自动拾取 |
| 拆除己方建筑 | 右键点击建筑 |
| 选中 / 聚焦 | 左键点击单位或建筑 |

## 打法建议

1. **首日铺经济**：采胶器 ×2 → 用宝藏点的奖励 → 视情况补一两个箭塔
2. **主动探索**：D2 起每晚刷新虫巢，靠侦察兵或探照灯驱散迷雾找出来打掉
3. **诱饵策略**：把瞭望塔 / 探照灯 / 侦察兵放在外围吸引虫子，保护核心和采集器
4. **天赋路径**：常见三种 build —— 进攻流（锋利→巢穴克星→扫击精进）/ 经济流（采胶丰收→坚固结构→紧急维修）/ 视野流（视野→远征→望远千里→灯火通明）
5. **血月预备**：D3 / D6 dusk 前攒满胶 + 修满前线建筑 + 把英雄拉回核心附近；血月夜虫子 hp +40% / dmg +30%、上限 ×2.2
6. **决战**：D6 dawn 终极 BOSS 出现后立即英雄出征，配合所有箭塔/法师塔围攻；终极 BOSS 直奔核心、自带永久揭雾，不会迷路

## 数值调优

所有数值集中在 `src/config.js`，重点关注：

- `phaseDurations` 各阶段时长（白天 75 / 傍晚 12 / 夜晚 60 / 黎明 10）
- `bloodMoonDays` / `winRequiredBloodMoons` 血月节奏与胜利条件
- `bloodMoon.bugHpMul / bugDamageMul / bugCapMul` 血月夜加压
- `terminalBoss.hp / damage` 终极 BOSS 强度
- `nest.bugCapByDay` 每巢虫上限阶梯
- `evolution.*` 进化因子曲线
- `talents.defs` 24 个天赋节点
- `aggro.rangeMulWhenAggroVisible` 仇恨范围倍数
- `treasures.rewardPool` 探索奖励权重
- `timeScales` 时间加倍档位
- `shop` 商店直购价格
- `fog` / `searchlight` 迷雾与探照灯参数
